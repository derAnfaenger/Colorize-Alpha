Colorize-Alpha

Most image manipulation programs implement a feature for colorizing a color to alpha, but its very hard to find ways to reverse this process; TO COLORIZE THE ALPHA PART OF AN IMAGE. This little tool is exactly built to do that. Features:

  1. standalone, portable program witout installation
  2. easy to use
  3. no ads
  4. does exactly what it has to do
  5. supports all common image file formats like jpg, png...
